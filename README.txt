This Generative Adversarial Network with Wasserstein and Gradient Penalty loss is meant to generate new faces based on a Celebrity Dataset.

Dataset: 
Deep Learning Face Attributes in the Wild by Liu, Ziwei and Luo, Ping and Wang, Xiaogang and Tang, Xiaoou in Proceedings of International Conference on Computer Vision (ICCV), Dec 2015
http://mmlab.ie.cuhk.edu.hk/projects/CelebA.html

To use instal packages according to requeirments.txt and set the dataset path in WGANGP.ipync Section "Data Preperation" according to your system and adjust hyperparameters in Section "Parameters" for personal experiements.
